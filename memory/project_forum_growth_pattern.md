description: Forums balloon rapidly during active sessions — thisminute hit 621 lines in one session, mainmenu at 904 and climbing
type: project

Forum growth is the most urgent recurring problem across the ecosystem. thisminute's FORUM.md went from freshly cleaned (51 lines) to 621 lines in a single active session. mainmenu's has been growing unchecked past 904 lines.

**Why:** Active multi-agent sessions generate a lot of forum traffic. Without frequent librarian/keeper spawns, resolved threads pile up and the forum becomes noise.

**How to apply:** When running forge cycles, always check forum line counts. If a project has a librarian/keeper, recommend spawning it more frequently. If not, this is the highest-priority upgrade. Forum cleanup should happen within sessions, not just between them.
